Presents checkboxes as toggle switches. This is a purely CSS toggle switch.

Basic Example

Toggle switch is an alternative presentation of a checkbox.

The checkbox is wrapped in a div with the toggle-switch class. The checkbox itself has toggle-switch-checkbox class and remains hidden.

The label element has toggle-switch-label class and is declared for the checkbox. Inside the label element, there are two span elements, with the following classes:

  • toggle-switch-inner
  • toggle-switch-switch

Text Label

The toggle can have optional text label. To specify the label, add a label element with the toggle-switch-label-text class after the toggle-switch-label.

Optionally, different labels can be used for "on" and "off" state. Inside the toggle-switch-label-text label, specify the "on" and "off" text in a span in toggle-switch-on-label-text and toggle-switch-off-label-text respectively.
