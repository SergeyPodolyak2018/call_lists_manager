Defining areas using colour

Genesys Vision uses background color to apportion function to screen areas. Generally, a dark background should be used for tool bars and palettes, a light background for the main workspace. Areas within those backgrounds can be defined with subtle 1px keylines.

Using color to show condition

Genesys Vision uses color to convey conditional statistics, therefore color should never be used arbitrarily. Generally, you can use the following as a guide to each color and its meaning:

Condition Hex Value
Normal #555D66
Focus, Active #2E69DB

Alert, Danger #EA4F6B
Warning #F8A740
Neutral #203B73
