Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

Create lighter, secondary text in any heading with a generic <small> tag or the .small class.

There is also a Primary and Secondary Headings that can be used.

Normal Body copy

Ark's global default font-size is 12px, with a line-height of 1.66667 or 20 px. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

When Select Mode is in use, the CSS class select-mode must be specified in the enclosing HTML element.

Alternate elements

Feel free to use <b> and <i> in HTML5. <b> is meant to highlight words or phrases without conveying additional importance while <i> is mostly for voice, technical terms, etc.
