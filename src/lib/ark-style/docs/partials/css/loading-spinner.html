This is the spinner to indicate loading status. It is purely written in css.

Default Spinner

The default spinner is a 50px * 50px sized css animated spinner. It is commonlly used to login components or some other panels.

Spinner on Dark Background

It can also be used in a dark background.

Fast Spinner

The fast spinner class makes the animation finishes faster.

Input Spinner

The input spinner is a 20px * 20px sized css animated spinner. It should be used within a input text field.

Internet Explorer Compatibility

Since this is using CSS3 animation, it does not support IE9 and below.
